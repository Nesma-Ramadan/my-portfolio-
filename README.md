# Nesma — Portfolio

A personal portfolio for a Junior Front-End Developer & UI/UX Designer. Built with React and Vite.

## Features

- **Hero** — Name, title, tagline, and CTAs
- **About** — Personal journey from UI/UX to front-end development
- **Skills** — Front-end and UI/UX skills (no fake percentages)
- **UI/UX Projects** — Behance-based design case studies
- **Front-End Projects** — GitHub projects in chronological order
- **Timeline** — Learning and experience progression
- **Contact** — Email and social links

## Tech Stack

- React 18
- Vite
- Semantic HTML
- CSS (no framework)
- Responsive, mobile-first design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Customize

Edit `src/data/portfolioData.js` to update:

- Name, title, tagline
- About paragraphs
- Skills lists
- UI/UX and front-end projects
- Timeline items
- Contact email and social links

## Build for Production

```bash
npm run build
```

Output will be in the `dist` folder.
